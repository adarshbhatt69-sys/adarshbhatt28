### 📓 TaskJournal — Full-Stack Productivity App

Designed and shipped a 10-endpoint REST API (FastAPI + async Motor) covering task CRUD, subtask management, completion tracking, bulk-clear, and server-side 7-day analytics using concurrent MongoDB aggregation pipelines (`asyncio.gather`). Built Pydantic v2 schemas with `@field_validator` (whitespace stripping) and `@model_validator` (cross-field due-date validation), plus `TaskOut.from_mongo()` for clean ObjectId serialization. Architected Expo Router file-based navigation — tab layout, dynamic task detail route, and a slide-up modal form — with safe-area handling, haptic feedback, and keyboard-avoiding forms. Delivered an insights dashboard with a weekly completion bar chart, KPI strip, and server-computed category/priority analytics, verified end-to-end with 18/18 pytest cases passing (Motor fully mocked via `unittest.mock`).
